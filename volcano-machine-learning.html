Machine Learning

Volcano Machine Learning Classification

This project applies machine learning methods to volcanic hazard data. It explores how clustering and classification can be used to identify structure within environmental datasets.

Research Question

Can machine learning methods identify meaningful groupings or classifications within volcanic hazard data?

This project applies machine learning methods to volcanic hazard data. It explores how clustering and classification can be used to identify structure within environmental datasets.

Data

  • Volcano attribute data
  • Explosivity indicators
  • Geographic and physical variables
  • Prepared modelling features

Method

  • Prepared features for modelling
  • Applied clustering to explore natural groupings
  • Used classification methods to test predictive structure
  • Evaluated model outputs and visualised results
  • Considered the limitations of applying machine learning to complex hazard data
