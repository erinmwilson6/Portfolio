ERIN WILSON PORTFOLIO

How To Open
1. Unzip the folder.
2. Double click index.html.
3. Use the navigation links to view the portfolio.

How To Edit Wording
1. Open the folder in Visual Studio Code.
2. Open the page you want to edit, for example index.html or projects.html.
3. Press Command and F on Mac to search for the sentence.
4. Change the wording.
5. Save the file.
6. Refresh the browser.

How To Add Your CV
1. Save your finished CV PDF inside this folder.
2. Rename it Erin Wilson CV.pdf.
3. Open cv.html.
4. Replace href="#" with href="Erin Wilson CV.pdf".
5. Remove the disabled class from the CV button.

Design Notes
This version avoids job application language and presents the work as a clean academic and professional portfolio.
Images are set to show fully rather than being cropped.
The colour palette is light and readable, with an optional dark mode.
