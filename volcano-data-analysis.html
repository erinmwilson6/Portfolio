Natural Hazards

Volcanic Hazard Data Analysis

This project uses exploratory data analysis to examine relationships between volcano characteristics, explosivity and reported impacts. It presents natural hazard data through charts and visual summaries.

Research Question

What patterns can be identified between volcano type, explosivity and reported fatalities?

This project uses exploratory data analysis to examine relationships between volcano characteristics, explosivity and reported impacts. It presents natural hazard data through charts and visual summaries.

Data

  • Volcano attribute data
  • Volcanic Explosivity Index values
  • Fatality and impact indicators
  • Geographic variables

Method

  • Cleaned and structured the dataset for analysis
  • Created summary statistics and visual outputs
  • Explored relationships between volcano type, VEI and reported impacts
  • Used charts to communicate patterns clearly
  • Reflected on the limitations of historical hazard records
