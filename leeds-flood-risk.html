Environmental Risk

Leeds Flood Risk And Social Vulnerability

This project explores how socially vulnerable neighbourhoods in Leeds spatially align with Environment Agency flood monitoring stations and flood risk communication areas. The work combines environmental risk information with deprivation data to show how web mapping can support clearer spatial interpretation.

Research Question

How do socially vulnerable neighbourhoods in Leeds align with flood monitoring infrastructure and flood risk communication areas?

This project explores how socially vulnerable neighbourhoods in Leeds spatially align with Environment Agency flood monitoring stations and flood risk communication areas. The work combines environmental risk information with deprivation data to show how web mapping can support clearer spatial interpretation.

Data

  • LSOA boundaries
  • Index of Multiple Deprivation
  • Environment Agency flood monitoring stations API
  • Environment Agency flood areas
  • Flood Map for Planning Flood Zones

Method

  • Prepared and projected spatial datasets for consistent analysis
  • Filtered Environment Agency API data to the Leeds study area
  • Joined deprivation indicators to neighbourhood boundaries
  • Mapped flood zones, stations and flood communication areas
  • Built an interactive Folium map with layered controls and tooltips

Output

Outcome

The project produced both static visualisations and an interactive web map. The strongest output is the layered map, which allows flood areas, flood zones, monitoring stations and deprivation patterns to be explored together.
